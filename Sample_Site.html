Data Structures

I would describe Data Structures as the building blocks for abstract data types, and stores the collection of data.

Examples of Data Structures

  • Stacks
  • Queues
  • Trees

Abstract Data Types would be what the user can view and allows the user to use the collection of data from the Data Structures.

Stacks

  1. Pop = Remove the top item on the stack
  2. Push = Put an item on the top of the stack
  3. Top = Discover the value(s) of the item on the top of stack
  4. Is Empty = Check if the stack has no items on the stack
